# ImageForCmd_GUI
이미지, 영상 문자화

## 설명
* OpenCV, tkinter 사용
* 픽셀마다 알맞은 문자를 적용, 시각적으로 변환시켜줌
* 친구들이 직접 사진을 바꿔볼수 있도록 Gui로 만듬
